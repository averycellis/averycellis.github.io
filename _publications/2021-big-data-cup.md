Wrote Markov expected threat model in R to analyze offensive value added by hockey players. Utilized novel dataset to evaluate player skill in non-traditional offensive categories like puck possession, skating, and passing. Combined discrete Markovian evaluation with contextual game state analysis to comprehensively measure offensive skill.

**Co-author:** Matthew Hurley

**Award:** Honorable Mention, Undergraduate Category
